# Sentinex
Sentinex is an AI-powered email security system that detects and classifies spam and phishing emails using NLP, Machine Learning, and Deep Learning. It analyzes email content in real time, identifies malicious threats, and helps protect users from fraud, credential theft, and cyberattacks, ensuring safer digital communication.

# AI-Based Spam and Phishing Detection in Emails

## Overview
This project detects spam and phishing emails using Machine Learning and Deep Learning techniques.

## Technologies Used
- Python
- Scikit-learn
- XGBoost
- LightGBM
- LSTM
- Flask
- HTML/CSS/JavaScript

## Features
- Spam Detection
- Phishing Detection
- Real-time Email Analysis
- TF-IDF Feature Extraction
- Hybrid ML + DL Model
